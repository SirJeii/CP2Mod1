Basic Document Structure

Every HTML document has a fundamental structure that contains the entire code.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Visible content goes here.
  </body>
</html>

Paragraphs & Line Breaks

The <p> element creates paragraphs, while the <br /> tag creates a line break.

<p>This is a paragraph.</p>
<p>This line contains <br /> a line break.</p>

Example Output

This is a paragraph.

This line contains
a line break.

Headings, Lines & Comments

HTML provides six levels of headings, from <h1> to <h6>, for section titles.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

Example Output

Heading 1

Heading 2

Heading 3

The <hr /> tag is used to create a thematic break or horizontal line.

<p>First paragraph.</p>
<hr />
<p>Second paragraph.</p>

Example Output

First paragraph.

Second paragraph.

Comments are not displayed by the browser but are useful for documenting your code.

<!-- This is a comment and will not be visible. -->

Example Output

(Nothing is displayed)

HTML Attributes

Attributes provide additional information about an element. The align attribute, for example, specifies horizontal alignment.

<p align="center">This text is centered.</p>

Example Output

This text is centered.
