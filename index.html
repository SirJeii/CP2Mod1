HTML Fundamentals

An overview based on your learning modules.

What is HTML?

HTML stands for Hypertext Markup Language. It's the standard language used to create and design web pages. Unlike a programming language, which performs functions, a markup language uses tags to identify and structure content.

For example, the <p> tag is used to define a paragraph:

<p>I'm a paragraph</p>

Basic Document Structure

Every HTML document has a fundamental structure that contains the entire code.

<!DOCTYPE html>
<html>
  <head>
    <!-- Metadata and scripts go here -->
    <title>Page Title</title>
  </head>
  <body>
    <!-- Visible content goes here -->
  </body>
</html>
  • <html>: The root element that wraps all the content on the page.
  • <head>: Contains meta-information about the document, like the title, character set, styles, and scripts. This content is not displayed on the page itself.
  • <body>: Contains all the visible content of the web page, such as headings, paragraphs, images, and links.

Paragraphs & Line Breaks

The <p> Element

The <p> element is used to define a paragraph of text. Browsers automatically add some space (a margin) before and after each paragraph.

The <br /> Element

The <br /> tag inserts a single line break. It's an empty tag, meaning it doesn't need a closing tag. It's useful for writing addresses or poems where the division of lines is important.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
<p>This is a line. <br /> And this is another line.</p>

Headings, Lines & Comments

HTML Headings (<h1> - <h6>)

HTML provides six levels of headings. <h1> is the most important (and largest), and <h6> is the least important (and smallest).

Horizontal Lines (<hr />)

The <hr /> tag creates a thematic break in the content, often displayed as a horizontal line. It's used to separate content.

HTML Comments

Comments are not displayed in the browser but can help document your HTML source code. They are useful for leaving notes or reminders for yourself or other developers.

<!-- Your comment goes here -->

Text Formatting

HTML has several elements for defining text with a special meaning.

<b> - Bold text

<strong> - Important text (semantically important)

<i> - Italic text

<em> - Emphasized text (semantically important)

<sub> - H2O (Subscript)

<sup> - E=MC2 (Superscript)

<ins> - Inserted text (underlined)

<del> - Deleted text (strikethrough)

HTML Attributes

Attributes provide additional information about HTML elements and are always specified in the start tag. They usually come in name/value pairs like name="value".

The `align` Attribute

The align attribute specifies the horizontal alignment of text. (Note: In modern HTML, CSS is preferred for styling and alignment).

<p align="center">
  This text is aligned to the center.
</p>

Attribute Measurements

Some attributes, like width, can take fixed values (e.g., "50px") or percentage values (e.g., "50%"). Percentages are relative to the parent container.

<!-- Fixed width -->
<hr width="100px" />

<!-- Percentage width -->
<hr width="50%" />
